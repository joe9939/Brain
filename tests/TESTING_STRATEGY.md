# Brain-Agent Circuit Testing Strategy

> **Status**: Implementing initial 11-circuit test suite for `.opencode/skills/brain-master.md`
> **Framework**: `tests/runner.js` (JS-based, grep/parse brain-master.md, return `{passed, message, time_ms}`)
> **Evidence**: `.omo/evidence/<circuit-name>.md` (auto-generated by each test)
> **Imperative**: No human judgment. Every test is fully automated and agent-executable.

---

## Architecture of the test suite

```
tests/
├── TESTING_STRATEGY.md       ← this file — the complete testing strategy
├── runner.js                 ← test runner (loads *.test.js from category dirs)
├── config.js                 ← shared paths (SKILL_FILE = brain-master.md)
├── circuits/                 ← circuit-level tests (one per mechanism)
│   ├── l1-pathway.test.js            [existing]
│   ├── l2-pathway.test.js            [existing]

│   ├── circuit-mood-layers.test.js   [new]
```

### Three validation tiers

| Tier | Method | What it checks | Cost |
|------|--------|----------------|------|
| **grep** | `content.includes()`, `content.match(regex)`, string ops | Section exists, key terms present, structural patterns | ~5ms per test |
| **simulation** | Parse pseudo-code logic, trace state transitions, verify formula consistency | Circuit logic correctness, data flow, boundary behavior | ~10ms per test |
| **functional** | Run node script that calls actual MCP tools via child_process | Tools respond correctly, data flows end-to-end | ~1-5s per test |

**Recommendation**: Use grep for sections-exist, simulation for logic consistency, functional for tool integration. All three are "agent-executable" — no human reads output.

## Evidence format

Every test writes an evidence file to `.omo/evidence/<name>.md` with this structure:

```markdown
# Circuit: <name>
**Status**: PASS | FAIL
**Timestamp**: <ISO>
**Duration**: <ms>ms

## Checks
- [x] Check 1: passed
- [x] Check 2: passed
- [ ] Check 3: FAILED (expected X found Y)

## Failures
- Check 3: expected `GLOBAL_STATE.mood` in L1 preamble, not found

## Raw grep evidence
```
GLOBAL_STATE found at lines: 85, 94, 125, 201, 340
```
```

This structure is machine-parseable (simple markdown) and human-readable.

## Adding a new circuit test

```javascript
// tests/circuits/circuit-<name>.test.js
const fs = require('fs');
const path = require('path');
const config = require('../config');

module.exports = {
  name: 'CIRCUIT: <Display Name>',
  run: async () => {
    const start = Date.now();
    const content = fs.readFileSync(config.SKILL_FILE, 'utf8');
    const results = [];

    // Check 1: grep for key pattern
    results.push({ name: 'Section exists', pass: content.includes('## SECTION') });

    // ... more checks ...

    const passed = results.every(r => r.pass);
    const failed = results.filter(r => !r.pass).map(r => r.name);

    // Write evidence file
    const evidenceDir = path.join(config.BRAIN_AGENT_DIR, '.omo', 'evidence');
    if (!fs.existsSync(evidenceDir)) fs.mkdirSync(evidenceDir, { recursive: true });
    const evidence = [
      '# Circuit: <name>',
      '**Status**: ' + (passed ? 'PASS' : 'FAIL'),
      '**Timestamp**: ' + new Date().toISOString(),
      '**Duration**: ' + (Date.now() - start) + 'ms',
      '',
      '## Checks',
      ...results.map(r => '- [' + (r.pass ? 'x' : ' ') + '] ' + r.name),
      ...(failed.length > 0 ? ['', '## Failures', ...failed.map(f => '- ' + f)] : []),
    ].join('\n');
    fs.writeFileSync(path.join(evidenceDir, 'circuit-<name>.md'), evidence);

    return {
      passed,
      message: passed
        ? 'All ' + results.length + ' <name> checks passed'
        : 'Failed: ' + failed.join(', '),
      time_ms: Date.now() - start,
    };
  },
};
```
